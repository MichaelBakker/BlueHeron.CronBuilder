ParameterValueType
Enumeration of possible parameter value types. Valid values:
  • 0: Any
  • 8: A single integer value
  • 9: List of integers
  • 10: Range of integers
  • 12: Step of integers
  • 16: A single MonthOfYear value
  • 17: List of MonthOfYear values
  • 18: Range of MonthOfYear values
  • 20: Step of MonthOfYear values
  • 32: A single DayOfweek value
  • 33: List of DayOfweek values
  • 34: Range of DayOfweek values
  • 36: Step of DayOfweek values
Valid combinations:
  • Minute: 0,8,9,10,12
  • Hour: 0,8,9,10,12
  • Day: 0,8,9,10,12
  • Month: 0,8,9,10,12,16,17,18,20
  • WeekDay: 0,8,9,10,12,32,33,34,36
